# Screenshots Directory

This directory contains screenshots of the app running on different platforms.

## Required Screenshots

To resolve the warnings in README.md, please add the following image files to this directory:

1. **android-market-data.png** - Market data screen on Android emulator
   - Shows the app displaying crypto market data with prices and 24h changes
   - Green/red color coding for positive/negative changes

2. **ios-market-data.png** - Market data screen on iOS simulator
   - Shows the app displaying crypto market data with prices and 24h changes
   - Green/red color coding for positive/negative changes

## How to Add Screenshots

1. Take screenshots from your Android emulator and iOS simulator
2. Save them with the exact filenames: `android-market-data.png` and `ios-market-data.png`
3. Place them in this `screenshots/` directory
4. The images will automatically appear in the main README.md
